## DBの仕様

| テーブル名  | 目的                       | 備考                                                                | 
| :---------: | :------------------------: | :-----------------------------------------------------------------: | 
| users       | ユーザー管理               | メールアドレスをIDとして使用                                        | 
| countrie    | 国を管理                   | 国コード(大文字2字)をIDとして使用                                      | 
| areas       | 地域を管理                 |                                                                     | 
| colors      | 色を管理                   |                                     | 
| flag_colors | 国と国旗の色の対応付け     |                                                                     | 
| quizzes     | クイズ問題の管理           | 出題用にひらがなの国名のカラムを追加.                               | 
| quiz_results| ユーザーとクイズの対応付け |                                                                     | 
| bookmarks   | ブックマークを管理         |                                                                     |
